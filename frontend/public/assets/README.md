# Assets Folder

## Logo

Đặt file logo của bạn vào đây với tên `logo.png` hoặc `logo.svg`.

**Yêu cầu:**

- Tên file: `logo.png` hoặc `logo.svg`
- Kích thước khuyến nghị: 200x60px (hoặc tỷ lệ tương tự)
- Định dạng: PNG (nền trong suốt) hoặc SVG

**Cách sử dụng:**

1. Đặt file logo vào thư mục này: `frontend/public/assets/logo.png`
2. Component Logo sẽ tự động load file này
3. Nếu không tìm thấy file, hệ thống sẽ hiển thị logo CSS dựa trên mô tả

**Lưu ý:**

- File trong thư mục `public` có thể được truy cập trực tiếp qua URL: `/assets/logo.png`
- Không cần import, chỉ cần đặt file vào đúng vị trí
